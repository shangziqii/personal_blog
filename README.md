## 技术栈

- 基于react框架（Create-React-App）

- 使用scss
- 使用vw+vh+rem+flex响应式布局
- 引入Ant Design
- 使用react-router
- 使用axios
- 使用reduce

## 启动项目

`cd my-app` 快速进入项目目录

`npm start` 启动项目

 

## 初始化项目

### 创建项目

`npm i -g create-react-app` 全局安装 `create-react-app`

`npx create-react-app my-app --template typescript` 创建基于TypeScript的react脚手架
